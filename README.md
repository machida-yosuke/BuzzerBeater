PDC ブザービターのリポジトリ
====

## 事前に用意するもの
- node.js (version 5.0.0以上)

## 環境構築
```
# 開発に使うnpmパッケージをインストール
yarn

# または
npm install
```

## 使用言語

- HTMLテンプレート: [pug](http://jade-lang.com/)
- CSSメタ言語: [Sass(scss)](http://sass-lang.com/)
- Javascript: [ES2015(ECMAScript 6)](https://babeljs.io/docs/learn-es2015/)
- vuex
- vuex
